Extension Chrome AutoApply

Capturez automatiquement les informations des offres d'emploi

Qu'est-ce que l'extension AutoApply ?

L'extension Chrome AutoApply est un outil puissant qui vous permet de capturer automatiquement les informations des offres d'emploi que vous consultez sur les sites de recrutement populaires comme LinkedIn, Indeed, Welcome to the Jungle, et bien d'autres.

Télécharger l'extension

Comment installer l'extension

  1. Téléchargez le fichier autoapply-extension.crx en cliquant sur le bouton ci-dessus.
  2. Ouvrez Chrome et accédez à chrome://extensions/ en tapant cette adresse dans la barre d'URL.
  3. Activez le Mode développeur en cliquant sur le bouton situé en haut à droite de la page.
  4. Glissez-déposez le fichier autoapply-extension.crx téléchargé dans la fenêtre des extensions.
  5. Confirmez l'installation lorsque vous y êtes invité en cliquant sur "Ajouter l'extension".
  6. Une fois installée, l'icône de l'extension apparaîtra dans la barre d'outils de Chrome.

Comment utiliser l'extension

Une fois l'extension installée, vous pouvez l'utiliser pour capturer les informations des offres d'emploi :

  1. Naviguez sur un site d'emploi et ouvrez une offre qui vous intéresse.
  2. Cliquez sur l'icône AutoApply dans la barre d'outils de Chrome.
  3. L'extension détectera automatiquement les informations de l'offre (titre, entreprise, lieu, description, etc.).
  4. Vérifiez les informations et cliquez sur "Enregistrer" pour ajouter l'offre à votre tableau de bord AutoApply.
  5. Accédez à votre tableau de bord AutoApply pour voir toutes vos candidatures enregistrées.
